# 认知诊断方法集

此存储库包含精选的认知诊断方法列表，包括其实现、相关研究论文以及每种方法的其他资源。

## 概述

认知诊断是一种用于分析和评估个体在特定知识或技能领域中优势和劣势的方法。它常用于教育评估中，以识别学生的学习需求并提供定制的教学。

## 用法

要使用特定的认知诊断方法，请导航到该方法的目录并按照该方法子文件夹中提供的具体说明进行操作。
